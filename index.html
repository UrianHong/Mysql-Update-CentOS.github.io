如何在CentOS6.5系统中安装Mysql5.6

CentOS 6.5自带安装了MySQL 5.1，但5.1有诸多限制，而实际开发中，我们也已经使用MySQL 5.6，这导致部分脚本在MySQL 5.1中执行会出错。我们只能升级MySQL

Download this project as a .zip file Download this project as a tar.gz file

备份数据库，升级MySQL通常不会丢失数据，但保险起见，我们需要做这一步。输入命令：

mysqldump -u xxx -h xxx -P 3306 -p --all-databases > databases.sql

停止MySQL服务，输入命令：

service mysqld stop

卸载旧版MySQL，输入命令：

yum remove mysql mysql-*

移除命令执行后，可再看看是否有残余的mysql，输入命令：

yum list installed | grep mysql

如果有，可输入命令删除：

rum remove mysql－libs

下载安装最新的rpm文件

rpm -Uvh http://repo.mysql.com/mysql-community-release-el6-5.noarch.rpm

安装MySQL，输入命令：

yum install mysql-community-server

安装完成后，输入命令查看MySQL版本号：

mysql -V

启动MySQL，输入命令：

service mysqld start

(可选)如果是新安装的数据库，新建一个数据库密码

方法一：
先切换到mysql的安装目录的bin文件夹下
在命令行中输入mysqladmin -u用户名 -p旧密码 password 新密码修改密码
方法二：
以root用户登录mysql命令行
输入以下数据库语句
use mysql;
UPDATE mysql.user SET password=PASSWORD(123456[你的新密码]) WHERE User='root';
FLUSH PRIVILEGES;
quit;
